Introducción

Es realmente importante, entender el contexto de lo que vamos a trabajar y para eso es importante entrar en contexto sobre JavaScript: ¿Que es?, ¿Para que sirve?, y ¿Por qué existe?. Esto es para facilitar el entendimiento de el tema a trabajar, ya que facilita la estructuración o encapsulamiento de la información en nuestra memoria.

¿Que es JavaScript?

JavaScript es un lenguaje programación que permite la creación de un objeto sin definir primero su clase, es decir, se basa en prototipos. Este es un leguaje multiparadigma, de un solo hilo, dinámico, con soporte para programación orientada a objetos, imperativa y declarativa.

¿Para qué sirve JavaScript?

Al ser este un lenguaje multiparadigma, se puede clasificar en programación funcional, programación basada en prototipos, imperativo e interpretativo.

Esto para entender que este lenguaje nos permite trabajarlo como un herramienta muy versatil en diferentes areas de desarrollo como por ejemplo: Desarrollo Web Frontend o Desarrollo Web Backend y algunos usos externos a la web como aplicaciónes de escritorio (mayoritariamente widgets).

¿Por qué existe JavaScript?

Este lenguaje de programación nace de la necesidad de hacer interactiva una página web. Ya que anteriormente las páginas eran solo informativas ya que no permitian ningun tipo de interacion activa por parte del usuario.

Elementos de un lenguaje de programación

En un lenguaje de probramación se necesitan dos componentes principales: datos y tareas.

Los datos es la información que se guardara en memoria y esta puede tener varios tipos como: números, strings (texto), boleanos, null o undefined(valores vacios), arrays, objetos, variables.

Las tareas son funciones que le asignamos al codigo para que haga algo con dichos datos.

¿Cómo se hace esto en JavaScript?

Tipo números: 28,35, 13...

Tipo string: "Cualquier tipo de texto", "4".

Tipo boleanos: true o false.

Valores vacios: null tipo null o undefined tipo undefined.

Arrays: ["texto", 4, "5", 64] tipo objeto.

Tipo objetos: {nombre : "tuNombre"}

Variables: var nombreDeLaVariable = valoreDeLaVariable;.

Hoisting

El concepto de Hoisting fue pensado como una manera general de referirse a cómo funcionan los contextos de ejecución en JavaScript (específicamente las fases de creación y ejecución).

Una estricta definición de hoisting sugiere que las declaraciones de variables y funciones son físicamente movidas al comienzo del código, pero esto no es lo que ocurre en realidad.

Lo que sucede es que las declaraciones de variables y funciones son asignadas en memoria durante la fase de compilación, pero quedan exactamente en dónde las has escrito en el código.

Ejemplo:

Al correr los siguientes 2 codigos se puede observar que la variable esta definida despues de ser llamada. Pero aun asi JavaScript funciona por su forma de interpretar.

Codigo 1

  console.log(miNombre);
  var miNombre = "nombreCualquiera";

Codigo 2

  hey();
  function hey() {
    console.log("Hola" + miNombre);
  }

Coherción

Coherción implicita

La Coherción implicita modifica la naturaleza del valor mediante el uso de un operador.

Ejemplo 1

  var = a 4 + "7" ;
  a = "47"

La naturaleza o tipo de a será string.

Ejemplo 2

  var b = 3 * "4";
  b = 12

La naturaleza o tipo de b será número.

Coherción explicita

La Coherción explicita modifica la naturaleza del valor mediante el uso de una funcion asignada.

Ejemplo 1

  d = 34;
  var e = String(d);
  e = "34"

La naturaleza o tipo de d es número, mientras que la de e será string.

Ejemplo 2

  f = "45";
  var g = Number(f);
  e = 45

La naturaleza o tipo de f es string, mientras que la de g será número.

Truthy & Falsy

El valor pasado como primer parámetro se convierte en un valor booleano, si es necesario. Si el valor se omite o es 0, -0, null, false, NaN, undefined, o la cadena vacía (" "), el objeto tiene un valor inicial de false.

Todos los demás valores, incluido cualquier objeto, un arreglo vacío ([ ]) o la cadena "false", crean un objeto con un valor inicial de true.

Valores false

  Boolean (0);   // return false
  Boolean (null);   // return false
  Boolean (NaN);   // return false
  Boolean (undefined);   // return false
  Boolean (false);   // return false
  Boolean ("")   // return false;

Valores true

  Boolean (" ");   // return true
  Boolean ("a");   // return true
  Boolean (1);   // return true
  Boolean ([ ]);   // return true
  Boolean ({});   // return true
  Boolean (true)   // return true;

Condicionales

if, else if, else
Los condicionales son un punto de infleccion del codigo en el cual tiene que entrar a decidir, si cumple una condición (condicionUno), entonces ejecute {ejecutarOrdenUno}.

  if (condicionUno) {
    ejecutarOrdenUno
  }

Si no se cumple la condicion ( ) anterior pero se cumple la siguente condición (condicionDos), entonces ejecute {ejecutarOrdenDos}. Nota es posible utilizar tantos else if como sea necesario.

  else if (condicionDos) {
    ejecutarOrdenDos;
  }

Si no se cumple ninguna de las condiciones ( ) anteriores, entonces ejecute {ejecutarOrdenTres}.

  else {
    ejecutarOrdenTres;
 }

Operador Ternario
El operador condicional (ternario) es el único operador en JavaScript que tiene tres operandos. Este operador se usa con frecuencia como atajo para la instrucción if.

  condition ? true : false;

Se evaluará la condición condition por medio de true o false y como respuestra entregará: true o false.
Ejemplo

  var numA = 1;
  var numB = 2;
  numA == numB ? "numA igual numB":"numA diferente numB"

El resultado a este codigo será: "numA diferente numB

Switch
La declaración switch evalúa una expresión ( ), comparando el valor de esa expresión con una instancia case, y ejecuta declaraciones { } asociadas a ese case, o ejecurara un default en caso de que ningun case se ejecute.

  var numero = 33;
  switch (numero){
    case 1:
      console.log("Soy uno!");
    break;
   caso 33:
      console.log("Soy un 33!");
    break;
    caso 50:
      console.log("Si, soy un 50");
    break;
    default:
      console.log("No soy nada");
  }

Metodos de Multiple Criterio
En este caso diferentes case evaluaran la misma declaración { } es decir, compartiran la misma declaración { }.

  var Animal = 'Loro';
  switch (Animal) {
    case 'Tucan':
    case 'Guacamaya':
    case 'Aguila':
    case 'Golondrina':
      console.log('Este animal es volador.');
    break;
    case 'Perro':
    case 'Vaca':
    default:
      console.log('Este animal no es volador.');
  }

Nota: en caso de que no se ponga un break; en el condicional switch se ejecutaran todas las declaraciones { } de los diferentes case hasta encontrar un break o hasta que evalue todas las declaraciones { } de los case existentes hasta el final.

Arrays

El objeto Array de JavaScript es un objeto global que es usado en la construcción de arrays, que son objetos tipo lista de alto nivel.

Los arrays son objetos similares a una lista cuyo prototipo proporciona métodos para efectuar operaciones de recorrido y de mutación. Tanto la longitud como el tipo de los elementos de un array son variables.

Creación de un array

  var frutas = ["manzana", "plátano", "cereza", "fresa"];
  console.log(frutas.length)

Es importante mencionar que la posición de los arrays se cuenta a partir de 0. También debe tenerse en cuenta que para saber la longitud de un array debe utilizarse nombreDelArray.length.


Acceso a un array
Ejemplo 1:

  let segundaFruta = frutas[1]
  segundaFruta = plátano

Ejemplo 2:

  let ultimaFruta = frutas[frutas.length - 1]
  ultimaFruta = fresa

Añadir un elemento al final del array

  var frutas = ["manzana", "plátano", "cereza", "fresa"];
  var masFutas = frutas.push("mora")
  frutas = ["manzana", "plátano", "cereza", "fresa", "mora"]
  masFrutas = 5   // tamaño del array en ese momento

Es importante reconocer que al utilizar nombreDelArray.push(" "). Esto modi el array al que se le aplicó el nombreDelArray.push(" ").

Elimina el último elemento del array

  frutas = ["manzana", "plátano", "cereza", "fresa", "mora"]
  var menosFutas = frutas.pop()
  frutas = ["manzana", "plátano", "cereza", "fresa"]
  menosFrutas = "mora"

Cabe mencionar que el nombre de la variable asignado menosFrutas, esta variable guarda el elemento del array eliminado.

Añade un elemento al inicio del Array

  frutas = ["manzana", "plátano", "cereza", "fresa"]
  var inicioFutas = frutas.unshift("papaya")
  frutas = ["papaya","manzana", "plátano", "cereza", "fresa"]
  inicioFrutas = 5

Cabe mencionar que el nombre de la variable asignado menosFrutas, esta variable guarda el elemento del array eliminado.

Elimina el primer elemento del array

  frutas = ["papaya","manzana", "plátano", "cereza", "fresa"]
  var borrarFutas = frutas.shift()
  frutas = ["manzana", "plátano", "cereza", "fresa"]
  borrarFrutas = "papaya"

Cabe mencionar que el nombre de la variable asignado borrarFrutas, esta variable guarda el elemento del array eliminado.

Posición del valor en un array

  frutas = ["manzana", "plátano", "cereza", "fresa"]

Objetos

¿Qué son los objetos?
Un objeto es una colección de datos relacionados y/o funcionalidad (que generalmente consta de algunas variables y funciones, que se denominan propiedades y métodos cuando están dentro de objetos).
Creación de un Objeto

  var persona = {
    nombre: ['Bob', 'Smith'],
    edad: 32,
    genero: 'masculino',
    intereses: ['música', 'esquí']
  };

Funciones dentro de un Objeto
Las funciones dentro de un objeto se definen con el nombre de la funcion nombreDeLaFuncion seguido de : function() { , aqui dentro se ingresan las variables, condiciones o ejecutables que se busque con la función. Es muy importante no olvidar cerrar la función} .

  var persona = {
    nombre: ['Bob', 'Smith'],
    edad: 32,
    genero: 'masculino',
    intereses: ['música', 'esquí']
    saludo : function () {
      alert('Hola, soy' + this.nombre[0] + '.');
    }
  };

Nota: es reelevante observar que la guncion a utilizar this.nombre el this. hace referencia a el objeto en el que se encuentra, nombre se refiere al elemento dentro del mismo objeto y [0] hace alución a la posición 0 del elemento nombre dentro del objeto persona.

Llamado de algún valor dentro del objeto

  persona.nombre   // Mostrará los valores que tengan pertenezcan a nombre
  persona.nombre[0]   // Mostrará el valor asignado en la posición 0
  persona.edad   //Mostrará el valor que pertenezca a edad dentro del objeto
  persona.intereses[1]   // Mostrará el valor que pertenezca a interes y se encuetre en la posición 1
  persona.saludo()   // ejecutara la función saludo, pentro del objeto persona

Bucles o loops

Los bucles ofrecen una forma rápida y sencilla de hacer algo repetidamente. Hay muchos diferentes tipos de bucles, pero esencialmente, todos hacen lo mismo: repiten una acción varias veces.

Los diversos mecanismos de bucle ofrecen diferentes formas de determinar los puntos de inicio y terminación del bucle. Hay varias situaciones que son fácilmente atendidas por un tipo de bucle que por otros.

For
Un ciclo for ejecuta una acción si y se repite mientras sea true una condición hasta que una condición especificada se evalúe como false.

  for ( exprIn; exprCond; exprAct) {
    evaluador
  }

NOTA:En los bucles for, suele parecerse a algo como: exprIn --> i = 0, exprCond --> i < numeroDeterminado y exprAct --> i++ .

While Una declaración while ejecuta sus instrucciones siempre que una condición especificada se evalúe como true.

  while (exprCond){
    evaluador
  }

Un ejemplo de este caso seria:

  let n = 0;
  let x = 0;
  while (n < 3) {
    n++;
    x += n;
  }

En este caso el bucle se ejecutará 3 veces ya que el final de:
Primera ejecución: n = 1 y x = 1
Segunda ejecución: n = 2 y x = 3
Tercera ejecución: n = 3 y x = 6
Para este punto n = 3, por lo tanto no se cumple la condición n < 3 , por lo tanto no realizara la ejecución.

For..in, For..of
La declaración for...in y for...of crean un bucle que se repite sobre objetos iterables.
La diferencia entre un bucle for...in y un bucle for...of.
Mientras que for...in itera sobre los nombres de propiedad, for...of itera sobre los valores de propiedad:

  const array = [3, 5, 7];
  array.foo = 'hola';
  for (let i in array) {
    console.log(i); // retornará "0", "1", "2", "foo"   }
  for (let i of array) {
    console.log(i); // entregará 3, 5, 7
  }

Debe resaltarse que al llamar el array array este entregará: [3, 5, 7, foo:"Hola"]
Es por esto que de la posición 3 del array array no entrega nada, es porque en la posicón 3 de array no se encuentra un valor, se encuenta un objeto.
Por lo que es interesante mencionar que el se pueden tener objetos dentro de un array.

Función constructora

La función constructora es la versión de JavaScript de una clase. Esta tiene todas las características que se espera de una función, aunque no devuelve nada o crea explícitamente un objeto, es decir, sólo define propiedades y métodos.
Se observa que la palabra clave this se usa aquí también — es básicamente decir que cuando se crea una de estas instancias de objeto, las propiedades marca, modelo y annio del objeto será igual al valor del nombre pasado a la llamada del constructor, y el método greeting() usará también el valor del nombre pasado a la llamada del constructor.

  function Auto(marca, modelo, annio){
    this.marca = marca;
    this.modelo = modelo;
    this.annio = annio;
  }

Uso de la función constructora

  var autoNuevo = new auto("Tesla", "Model 3", "2020");
  var autoNuevo2 = new auto("Tesla", "Model X", 2018);
  var AutoNuevo3 = new auto("Toyota", "Corolla", 2020);

En este caso autoNuevo, autoNuevo2 y AutoNuevo3 son objetos independientes que utilizaron la estructura de la misma función constructora.
